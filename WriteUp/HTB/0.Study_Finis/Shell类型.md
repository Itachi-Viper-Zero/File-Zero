| Shell 类型        | 通信方式                                          |
| --------------- | --------------------------------------------- |
| `Reverse Shell` | 连接回我们的系统，通过反向连接给予我们控制权。                       |
| `Bind Shell`    | 等待我们连接到它，一旦我们连接成功，就给予我们控制权。                   |
| `Web Shell`     | 通过 Web 服务器进行通信，通过 HTTP 参数接受我们的命令，执行它们，然后返回输出。 |
